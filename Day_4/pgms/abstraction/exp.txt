Simple Explanation

Abstraction means:

    Show only important information and hide unnecessary details.

User knows WHAT to do but doesn't know HOW it works internally.

Real-World Example
Car

You know:

    Start Car
    Accelerate
    Brake

You don't need to know:

    Fuel Injection
    Piston Movement
    Combustion Process

Those details are hidden.

This is Abstraction.

Simple Program

    #include <iostream>
    using namespace std;

    class Car
    {
    public:
        void start()
        {
            cout << "Car Started" << endl;
        }
    };

    int main()
    {
        Car c;

        c.start();

        return 0;
    }

Output
    Car Started


Student Understanding

    User sees:
    start()

    Hidden:
    Engine operations
    Fuel system
    Battery operations

That's Abstraction.

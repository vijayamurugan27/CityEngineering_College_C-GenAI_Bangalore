Friend Class
Simple Explanation

Instead of giving permission to one function,

we give permission to an entire class.

Real-World Example

A company allows:

HR Department

to access employee records.

Every HR employee gets access.

Similarly:

    Class B

can access private members of

    Class A

Simple Program

    #include <iostream>
    using namespace std;

    class A
    {
    private:
        int salary = 50000;

        friend class B;
    };

    class B
    {
    public:
        void show(A obj)
        {
            cout << "Salary = " << obj.salary;
        }
    };

    int main()
    {
        A a;
        B b;

        b.show(a);

        return 0;
    }

Output

Salary = 50000

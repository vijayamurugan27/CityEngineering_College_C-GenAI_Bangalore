Custom Exceptions

Simple Explanation

We can create our own error messages.

Real-World Example

College Rules

        Instead of generic errors:

                Error

        College says:

                Attendance below 75%

Specific message.

Program

        #include <iostream>
        using namespace std;
        class InvalidMarks
        {
        };
        int main()
        {
            try
            {
                int marks = -10;

                if(marks < 0)
                {
                    throw InvalidMarks();
                }
            }
            catch(InvalidMarks)
            {
                cout << "Marks Cannot Be Negative";
            }
            return 0;
        }

Output
Marks Cannot Be Negative

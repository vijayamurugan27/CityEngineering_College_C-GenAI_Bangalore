PARAMETERIZED CONSTRUCTOR

What is it?

    A constructor that accepts values.

Child Example

You order a toy car:

    Color = Blue
    Name = Racing Car

Factory builds exactly what you asked.

Example
        #include <iostream>
        using namespace std;

        class Student
        {
        public:
            string name;

            Student(string n)
            {
                name = n;
            }

            void show()
            {
                cout << "Student Name: " << name << endl;
            }
        };

        int main()
        {
            Student s1("Rahul");

            s1.show();

            return 0;
        }

    Output
        Student Name: Rahul

Understanding

    This value:

        "Rahul"

    goes into:

        Student(string n)

    Then:

        name = n;

    stores it.

Visual Flow
        Student s1("Rahul")
        ↓
        Constructor receives Rahul
        ↓
        name = Rahul

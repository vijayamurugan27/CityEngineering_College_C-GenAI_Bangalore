Understanding Step-by-Step

    Constructor
        ptr = new int;
            creates memory.

    Destructor
        delete ptr;
            cleans memory.


What happens if destructor is NOT used?

Answer:
        Memory Leak happens

What is Memory Leak?
        Unused memory remains occupied.
        Program wastes RAM.

FULL LIFE CYCLE OF OBJECT

        Object Created
        ↓
        Constructor Runs
        ↓
        Object Used
        ↓
        Object Destroyed
        ↓
        Destructor Runs

Easy Comparison Table
                
        Constructor	            Destructor

        Multiple allowed	    Only one destructor

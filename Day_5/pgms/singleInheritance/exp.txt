Single Inheritance
Simple Explanation

One child inherits from one parent.

Real-World Example
Father
   ↓
Son

Only one parent.

Diagram
Animal
   ↓
Dog
Program
#include <iostream>
using namespace std;

class Animal
{
public:
    void eat()
    {
        cout << "Animal Eats" << endl;
    }
};

class Dog : public Animal
{
};

int main()
{
    Dog d;

    d.eat();

    return 0;
}
Output
Animal Eats

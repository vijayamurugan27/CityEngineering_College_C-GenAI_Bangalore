without rewriting.

Real-World Example
Calculator

Can compare:

Integers
Decimals

using same logic.

Program
#include <iostream>
using namespace std;

template <class T>
T maximum(T a, T b)
{
    return (a > b) ? a : b;
}

int main()
{
    cout << maximum(10,20) << endl;

    cout << maximum(4.5,8.2);

    return 0;
}
Output
20
8.2

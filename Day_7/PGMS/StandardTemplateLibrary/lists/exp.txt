List
Simple Explanation

List stores elements in linked form.

Easy insertion and deletion.

Real-World Example
Train Compartments

Compartments are connected.

You can add or remove compartments easily.

Program
#include <iostream>
#include <list>

using namespace std;

int main()
{
    list<int> l;

    l.push_back(10);
    l.push_back(20);
    l.push_back(30);

    for(int x : l)
    {
        cout << x << " ";
    }
}
Output
10 20 30

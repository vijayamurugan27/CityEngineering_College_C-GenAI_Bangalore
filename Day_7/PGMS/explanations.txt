1. Dynamic Memory Allocation (new, delete)
Simple Explanation

Normally variables are created during compilation.

int num = 10;

Memory is automatically allocated.

Sometimes we don't know how much memory we need beforehand.

So we ask memory from the computer while the program is running.

This is called:

Dynamic Memory Allocation

using:

new
delete
Real-World Example
Hotel Room Booking

Imagine:

Fixed Rooms = Static Memory

Book Room When Needed
Release Room After Checkout
= Dynamic Memory
Program
#include <iostream>
using namespace std;

int main()
{
    int *ptr;

    ptr = new int;

    *ptr = 100;

    cout << *ptr << endl;

    delete ptr;

    return 0;
}
Output
100
Student Understanding
new
 ↓
Allocate Memory

delete
 ↓
Release Memory
2. Pointers to Objects
Simple Explanation

A pointer can store the address of an object.

Real-World Example

Imagine:

Student = House

Student Address = Pointer

Instead of going directly to the house,

we use the address.

Program
#include <iostream>
using namespace std;

class Student
{
public:
    string name;

    void display()
    {
        cout << name;
    }
};

int main()
{
    Student *ptr = new Student;

    ptr->name = "John";

    ptr->display();

    delete ptr;

    return 0;
}
Output
John
Student Understanding
Object Access

obj.name

Pointer Object Access

ptr->name

Arrow operator:

->

means:

Go to object through pointer
3. Array of Objects using Pointers
Simple Explanation

Instead of creating one object,

create multiple objects dynamically.

Real-World Example
Classroom
Student 1
Student 2
Student 3
Student 4

Many students.

One class.

Program
#include <iostream>
using namespace std;

class Student
{
public:
    int roll;
};

int main()
{
    Student *ptr;

    ptr = new Student[3];

    ptr[0].roll = 101;
    ptr[1].roll = 102;
    ptr[2].roll = 103;

    cout << ptr[0].roll << endl;
    cout << ptr[1].roll << endl;
    cout << ptr[2].roll << endl;

    delete[] ptr;

    return 0;
}
Output
101
102
103
Student Understanding
new Student[3]

Creates:

Student 1
Student 2
Student 3

5. Function Templates
Simple Explanation

Same function works for:

int
float
double
char

without rewriting.

Real-World Example
Calculator

Can compare:

Integers
Decimals

using same logic.

Program
#include <iostream>
using namespace std;

template <class T>
T maximum(T a, T b)
{
    return (a > b) ? a : b;
}

int main()
{
    cout << maximum(10,20) << endl;

    cout << maximum(4.5,8.2);

    return 0;
}
Output
20
8.2
Student Understanding
T

Can become:

int
float
double
char
6. Class Templates
Simple Explanation

Entire class becomes reusable.

Real-World Example
Storage Box

Same box design stores:

Books
Toys
Electronics

Different contents.

Same structure.

Program
#include <iostream>
using namespace std;

template <class T>
class Box
{
public:
    T value;

    void show()
    {
        cout << value << endl;
    }
};

int main()
{
    Box<int> b1;

    b1.value = 100;

    b1.show();

    Box<string> b2;

    b2.value = "Hello";

    b2.show();

    return 0;
}
Output
100
Hello
Student Understanding
Box<int>

Box<float>

Box<string>

Same class.

Different types.

8. Vector
Simple Explanation

A Vector is a dynamic array.

Size can grow automatically.

Real-World Example
WhatsApp Group

New members can join anytime.

Size keeps increasing.

Program
#include <iostream>
#include <vector>

using namespace std;

int main()
{
    vector<int> v;

    v.push_back(10);
    v.push_back(20);
    v.push_back(30);

    for(int x : v)
    {
        cout << x << " ";
    }
}
Output
10 20 30
Student Understanding
push_back()

Adds new element

10. Stack

12. Iterators
Simple Explanation

Iterator is used to move through container elements.

Real-World Example
TV Remote

Moves through channels one by one.

Program
#include <iostream>
#include <vector>

using namespace std;

int main()
{
    vector<int> v={10,20,30};

    vector<int>::iterator it;

    for(it=v.begin(); it!=v.end(); it++)
    {
        cout << *it << " ";
    }
}
Output
10 20 30
Student Understanding
begin()
    ↓
First Element

end()
    ↓
After Last Element
13. Basic Algorithms
Simple Explanation

STL provides ready-made algorithms.

No need to write complex logic.

Real-World Example
Calculator

Instead of manually calculating:

Addition
Sorting
Searching

Use built-in functionality.

Sorting Example
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main()
{
    vector<int> v={40,10,30,20};

    sort(v.begin(),v.end());

    for(int x:v)
    {
        cout<<x<<" ";
    }
}
Output
10 20 30 40
Student Understanding
sort()

Automatically sorts data

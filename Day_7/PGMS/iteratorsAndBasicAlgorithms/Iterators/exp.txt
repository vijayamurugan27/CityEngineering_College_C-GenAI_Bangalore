Iterators
Simple Explanation

Iterator is used to move through container elements.

Real-World Example
TV Remote

Moves through channels one by one.

Program
#include <iostream>
#include <vector>

using namespace std;

int main()
{
    vector<int> v={10,20,30};

    vector<int>::iterator it;

    for(it=v.begin(); it!=v.end(); it++)
    {
        cout << *it << " ";
    }
}
Output
10 20 30
Student Understanding
begin()
    ↓
First Element

end()
    ↓
After Last Element
